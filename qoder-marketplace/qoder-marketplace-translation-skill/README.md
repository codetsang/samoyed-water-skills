# Translation / 翻译

> 在不同语言之间翻译文本，兼顾文化适配、术语一致性与多种文档类型。

## 简介

本 skill 帮助 AI 在翻译时保留原意、语气与上下文，处理习语、文化差异、领域术语与正式程度。支持技术文档、营销文案、UI 字符串、法律文本与一般 prose，并按目标语言习惯输出。

## 适用场景

- 将内容翻译为目标语言（技术文档、营销、UI、法律等）
- 需要术语一致、文化本地化或多种正式程度
- 保留 Markdown、HTML、JSON 等格式与占位符的翻译任务
