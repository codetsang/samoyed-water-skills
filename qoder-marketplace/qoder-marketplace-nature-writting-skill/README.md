# Natural Writing / 自然写作

> 让 AI 写出更像人写的文字，减少套路化、可识别的「AI 味」。

## 简介

本 skill 针对大模型写作中常见的模式化问题（意义膨胀、AI 词汇堆叠、破折号滥用等），提供具体改写原则与自检清单，帮助生成更自然、更具体的 prose。

## 适用场景

- 撰写邮件、报告、文章、博客、营销文案、文档等面向人类的文字
- 用户要求「更自然」「少一点 AI 感」「更像人写的」
- 任何需要产出可读 prose 的任务
